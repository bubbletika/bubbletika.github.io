Python Project

Every week, I volunteer at Clairemont High School as an after school Programming Instructor to give highly motivated students their first taste of Computer Science. Currently, I'm teaching the basics of Python and create exercises for them to complete so they can learn how to write code.

I call this the Python Project because it is an attempt at improving my community's education system. Many schools don't offer the chance to practice coding, and I think that causes many people have misconceptions about the nature and culture of Computer Science. This project is a way to get students excited about the many wonders of Computer Science.
